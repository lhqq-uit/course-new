## Tenchnical stack: MEVN stack
- **Database**: MongoDB
- **Web Framwork**: ExpressJS
- **Frontend Framwork**: No
- **Platform**: NodeJS

## Setup
**Go to folder "config" and edit the files:** "database.js", "domain.js", "key.js"

## Online learning website.

1. Install 
```
    npm i
```
  
2. Run
```
    npm start
```
